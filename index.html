Sobre o KaIaK3 Server

O KaIaK3 Server é um projeto independente criado com o objetivo de disponibilizar conteúdo técnico gratuito para concursos técnicos contando com videos de questões resolvidas e textos didáticos, sempre se baseando em editais passados de concursos, especificamente da Petrobras.
